∞RVM 1.29.5 released

Posted at December 12, 2018 by Piotr Kuczynski tags: releases, rvm1

RVM 1.29.5 released with few features and bug fixes.

∞1.29.5

12 December 2018 - Full Changelog

New features:

  • Switch to new maven-based JRuby download URLs
  • RailsExpress patches for 2.3.8, 2.4.5 and 2.5.3 #4476
  • Add support for Void linux #4478
  • Add support for MINGW in main rvm script #4493
  • Add support for gcc@7 on macOS with Homebrew #4500
  • Add support for Trisquel linux #4478
  • Add fail warning when Avira antivirus running #4498

Bug fixes:

  • Allow HTTP 2.0 servers to be used for downloads
  • Remove too restrictive check for LLVM with TruffleRuby #4427
  • Fix trap restoration on Amazon Linux #4428
  • Fix Amazon Linux 2 detection #4435
  • Fix libssl dependency for Elementary 5.0 Juno #4448
  • Fix Ruby 2.5.0 and 2.5.1 build with LibreSSL 2.7.0 #4483
  • Fix libssl dependency for Mint 19 #4482
  • Use --no-ri or --no-document depending on ruby version #4492

Upgraded Ruby interpreters:

  • Add support for TruffleRuby 1.0.0-rc3 #4419, 1.0.0-rc5 #4440, 1.0.0-rc6 #4452, 1.0.0-rc7 #4466, 1.0.0-rc8 #4489, 1.0.0-rc9 #4489, and 1.0.0-rc10 #4512.
  • Add support for Ruby 2.3.8, 2.4.5, 2.5.2, 2.5.3 #4474, 2.6.0-preview3 #4490, and 2.6.0-rc1 #4513
  • Add support for JRuby 9.2.1.0 #4491, 9.2.2.0 #4495, 9.2.3.0 #4496, 9.2.4.0 #4504, 9.2.4.1 #4509, and 9.2.5.0 #4514
  • Add support for mruby 1.4.1 \4517 and 2.0.0 \4516

Binaries:

  • Ubuntu 18.04 x64 binaries for Ruby 2.3.7, 2.4.4 and 2.5.1 #4438
