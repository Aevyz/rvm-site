∞RVM 1.29.7 released

Posted at January 3, 2019 by Piotr Kuczynski tags: releases, rvm1

RVM 1.29.7 released with few features and bug fixes.

∞1.29.7

3 January 2019 - Full Changelog

New features

  • Set Ruby 2.6.0 as the default Ruby version #4544
  • RailsExpress patches for 2.6.0 #4547
  • Add support for gcc@8 with Homebrew and drop gcc < 4.9 #4556

New interpreters

  • Add support for Ruby 2.6.0-rc2 #4526 and 2.6.0 #4542

Bug fixes

  • Install libssl1.0.2 for old versions of Ruby on Kali linux #4522
  • Fix Linux Mint 19 dependencies #4524
  • Fix Mojave Command Line Tools version detection #4404
  • Fix return code in signature check function #4530
  • Check stable branch only for latest version #4534
  • Fix comparison of local and online stable versions #4539
  • Remove reference to keys.gnupg.net in favour of pool.sks-keyservers.net #4550
  • Fix Homebrew docs links #4551
  • Drop invalid link for osx-gcc-installer #4552
  • Use gcc from XCode for macOS Mojave (10.14) #4557
  • Stop uninstalling gcc from Homebrew #4558

Changes

  • Drop usage of bit.ly for link shortening #4555
